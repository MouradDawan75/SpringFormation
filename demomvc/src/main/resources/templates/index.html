Attributs de Thymeleaf

Le contenu des attributs Thymeleaf peut être du simple texte ou une expression qui sera interprétée par Thymeleaf à l'exécution. Les expressions sont délimitées par:


${...} : pour les expressions à évaluer
*{...} : pour les expressions de selection à évaluer
@{...} : pour les expressions d'url
~{...} : pour les expressions de fragment
